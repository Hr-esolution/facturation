# Dashboard Navigation Workflow

## User Journey Flow

```
Connexion utilisateur
         ↓
      Dashboard
         ↓
┌─────────────────────────┐
│        Buttons          │
├─────────────────────────┤
│ • Mon Entreprise        │ ← View/Manage company data
│ • Mes Documents         │ ← Access company documents  
│ • Mes Clients           │ ← Manage client information
│ • Paramètres Factures   │ ← Configure invoice settings
│ • Champs Factures       │ ← Customize invoice fields
└─────────────────────────┘
```

## Detailed Navigation Path

1. **Connexion Utilisateur** → Authentication/Login
2. **Dashboard** → Main application landing page
3. **Navigation Options**:
   - **Mon Entreprise**: For managing company data and settings
   - **Mes Documents**: For accessing and managing company documents
   - **Mes Clients**: For client management functionality
   - **Paramètres Factures**: For configuring invoice parameters
   - **Champs Factures**: For customizing invoice field configurations
